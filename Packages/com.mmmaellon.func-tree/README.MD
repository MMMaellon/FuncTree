## Overview

FuncTree is a visual scripting language that attempts to simply VRChat world creation for beginners with little to no coding experience. It's named FuncTree because it's based on the functional programming language Elm and what's commonly referred to as the Elm Architecture. Also each graph is a tree and each node is a function.

Differences from Udon
 - No messy graphs. Every graph flows top to bottom, left to right.
 - Perscriptive coding. It holds your hand more and enforces the "correct" way to code things.
